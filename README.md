# PolyHitboxes

![Compact Powered by OneConfig](https://polyfrost.org/img/compact_vector.svg)

A hitbox modification mod

## Features

- Toggle Keybind
- Show Condition - Always / Hovered / Toggled / Never
- Line Style - Normal / Proportioned / Dashed
- Hovered Color
- Sides
- Outline
- Eye Height
- View Ray
